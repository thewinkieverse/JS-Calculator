## 🌍 About This Project

This is **Project #02 inside The Winkieverse** — a fully functional **JavaScript Calculator** built using:

- HTML  
- CSS  
- Pure JavaScript (Vanilla JS)  

No frameworks.  
No libraries.  
Just logic, structure, and real-world problem solving.

This project focuses on building a strong understanding of **JavaScript logic, event handling, and calculations**.

---

## 🧮 Calculator Overview

The JavaScript Calculator allows users to:

- ➕ Perform addition  
- ➖ Perform subtraction  
- ✖️ Perform multiplication  
- ➗ Perform division  
- 🔢 Input numbers using buttons  
- 🧹 Clear the screen  
- ⌫ Delete last input  
- 🟰 Get calculated results  

---

---

## ⚙️ How the Logic Works (High Level)

### 1️⃣ Input Handling
- User clicks buttons (numbers/operators)
- Values are appended to the display

### 2️⃣ Expression Building
- Inputs are stored as a string expression  
- Example: "5 + 3 * 2"

### 3️⃣ Calculation
- When "=" is clicked:
  - Expression is evaluated
  - Result is displayed

### 4️⃣ Clear & Delete
- Clear (C): Resets the display  
- Delete (⌫): Removes last character
